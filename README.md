## 初始化

语言选择C++，预计不会用到强化学习框架。

核心决策应该会使用打分机制，给予每个条件一定的数值，比如能直接摧毁水晶数值记为极大，每次选择数值最大的方案之行。这样方便最终决策的统一性。
